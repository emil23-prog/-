## Описание программы
**Weather Diary** — это графическое приложение (GUI) на Python для ведения персонального журнала наблюдений за погодой. Программа решает проблему структурированного хранения данных о климатических изменениях.

**Основные функции:**
- **Добавление записей:** фиксация даты, температуры, осадков и текстового описания.
- **Фильтрация:** быстрый поиск записей с температурой выше заданного порога.
- **Надежность:** автоматическое сохранение всех данных в файл `weather_data.json` и их загрузка при старте.
- **Валидация:** проверка корректности ввода (температура должна быть числом, описание не может быть пустым).
